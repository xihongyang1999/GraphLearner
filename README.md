### Requirements

The proposed GraphLearner is implemented with python 3.8.8 on a NVIDIA 3090 Ti GPU. 

Python package information is summarized in **requirements.txt**:

- torch==1.8.0
- tqdm==4.61.2
- numpy==1.21.0
- tensorboard==2.8.0
